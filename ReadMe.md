# React Boilerplate

This is a vite Boilerplate for using in react projects

## Installation

Use the package manager [npm](https://www.npmjs.com) or [yarn](https://yarnpkg.com) to install dependencies.

```bash
npm i
```

Or

```bash
yarn install
```

## Running the app

```bash
npm run dev
```

or

```bash
yarn dev
```
